
## 1.  Amazon EC2 = VPS
Merupakan sebuah VPS / server tradisional, cocok dipakai untuk backend monolith dan legacy system 

## 2. AWS Lambda = Serverless
Layanan serverless milik AWS, user tanpa manage server, cocok di pakai untuk deploy api kecil atau micro service.
AWS Lamda itu sesimple platflom yang di gunakan untuk deploy single function 

## 3. Amazon ECS / Amazon EKS
- Untuk Docker / container
- ECS → lebih simple
- EKS → Kubernetes (enterprise banget)
## 4. Amazon S3
- Penyimpanan file (gambar, video, dokumen)
- Use case:
    - upload user
    - backup
    - static website

## 5. Amazon RDS
- MySQL / PostgreSQL / dll
- Dipakai untuk:
    - aplikasi bisnis
    - ecommerce
    - ERP

## 6. Amazon DynamoDB
- NoSQL (key-value)
- Super scalable
- Dipakai untuk:
    - high traffic apps
    - realtime system
